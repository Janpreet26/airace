Language

Lua is the scripting language used for this thing.

Credit

If you tweak someone else's script, you should provide the ID of that script.

parentScript()This should return the ID of the script you tweaked.

Example: 

function parentScript()
  return 3
  -- this script is a tweak of the script with ID 3.
end

Track

A track is represented as a sequence of nodes.

Events

Implement these functions to allow the program to drive the car around the track.

start()This is called only once when the race starts.

update()This is called once every tick of the race.

newSection()This is called when the car enters a new section of the track.

Actions

Use thecarobject to manipulate the car.

car:accelerate()The X coordinate of the car's position.

car:decelerate()The Z coordinate of the car's position.

car:turnLeft()The X coordinate of the car's position.

car:turnRight()The Z coordinate of the car's position.

Example: 

function update()
  car:accelerate()
  -- accelerate every tick
end

Car Info

Use thecarobject to get information about the car.

car.position.xThe X coordinate of the car's position.

car.position.zThe Z coordinate of the car's position.

car.speedThe speed of the car.

Track Info

Use thetrackobject to get information about the track.

track.lastNodeThe last node the car passed through

track.nextNodeThe next node the car must pass through.

track.nodeAfterThe node the car must pass through after passing track.nextNode.
